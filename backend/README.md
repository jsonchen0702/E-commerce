### APIs

Users endpoints:
- `/api/v1/user/create_account`
- `/api/v1/user/login`
- `/api/v1/user/logout`

Admin endpoints:
- `/api/v1/admin/create_account`
- `/api/v1/admin/login`
- `/api/v1/admin/logout`
- `/api/v1/admin/new_product`

Other endpoints:
- `/api/v1/products`
